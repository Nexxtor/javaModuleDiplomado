# Indicaciones
## Estructura básica

* Cree una clase Person, la cual contendrá los siguientes atributos:
    * Id -> String
    * Age -> int
    * Firstname -> String
    * Lastname -> String
    
    Todos los atributos deben tener tipo de acceso privado; además cada uno de ellos deben poseer su respectivo "setter" y "getter".
